# Bookers2

これはDMM WEBCAMPの課題でつくった本の感想投稿サービスです。

### 機能一覧

* ログイン
* 投稿
* コメント
* いいね
* フォロー
* 検索（ユーザー・投稿）

etc...

随時機能追加しています。


### 開発環境

* Ruby 2.6.3
* Ruby on Rails 5.2.4.3
* Bootstrap 4.5(Gem)
* jQuery(Gem)
* Font Awesome(Gem)
